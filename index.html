Display

La propiedad CSS display especifica el tipo de caja de un elemento. Define cómo se debe mostrar un elemento en la página.

Para añadir display a un elemento HTML, puedes utilizar el atributo style o definir una regla de estilo en una hoja de estilos CSS.

Por ejemplo, para hacer que un elemento sea un bloque, puedes añadir style="display: block;" al elemento HTML o definir en tu hoja de estilos:

elemento {
        display: block;
      }

Position

La propiedad CSS position especifica el método de posicionamiento de un elemento en la página. Puede tener los siguientes valores:

  • static: El elemento sigue el flujo normal del documento.
  • relative: El elemento se posiciona con respecto a su posición original.
  • absolute: El elemento se posiciona con respecto al primer ancestro posicionado.
  • fixed: El elemento se posiciona con respecto a la ventana del navegador.

Para añadir position a un elemento HTML, puedes utilizar el atributo style o definir una regla de estilo en una hoja de estilos CSS.

Z-index

La propiedad CSS z-index especifica el orden de apilamiento de un elemento en relación con otros elementos en la página. Cuanto mayor sea el valor de z-index, más arriba estará el elemento en la pila de capas.

Para añadir z-index a un elemento HTML, puedes utilizar el atributo style o definir una regla de estilo en una hoja de estilos CSS.

Overflow

La propiedad CSS overflow especifica qué hacer si el contenido de un elemento es demasiado grande para caber en su contenedor.

Los valores comunes para overflow son:

  • visible: El contenido sobresale del contenedor.
  • hidden: El contenido que no cabe se oculta.
  • scroll: Se agrega una barra de desplazamiento para permitir desplazarse al contenido.
  • auto: Se agrega una barra de desplazamiento solo si es necesario.

Para añadir overflow a un elemento HTML, puedes utilizar el atributo style o definir una regla de estilo en una hoja de estilos CSS.

Float and Clear

El uso de `float` en CSS es una técnica que se utiliza para posicionar elementos a la izquierda o a la derecha dentro de su contenedor. Es comúnmente utilizado para crear diseños de varias columnas o para colocar imágenes junto al texto.

Float:

La propiedad `float` se utiliza para cambiar el flujo normal de los elementos en una página web. Cuando aplicas `float` a un elemento, este se desplaza hacia la izquierda o hacia la derecha dentro de su contenedor, y los demás elementos del documento se reorganizan alrededor de él.

Clear:

La propiedad `clear` se utiliza para indicar que un elemento no debe colocarse junto a los elementos flotantes. Esto se usa comúnmente para evitar que un elemento se desplace junto a los elementos flotantes en un diseño.

Ejemplo:

Supongamos que tienes dos elementos `div` que deseas colocar uno al lado del otro. Puedes usar `float` para lograrlo. También puedes usar `clear` para asegurarte de que ningún elemento posterior se desplace junto a los elementos flotantes.

    
      <div class="izquierda">Contenido Izquierdo</div>
      <div class="derecha">Contenido Derecho</div>
      <div class="clearfix"></div>

En este ejemplo, los elementos con las clases `.izquierda` y `.derecha` se colocarán uno al lado del otro dentro de su contenedor. El `div` con la clase `.clearfix` se coloca después de los elementos flotantes y asegura que ningún elemento posterior se desplace junto a ellos.

Display: inline-block

La propiedad CSS display: inline-block permite que un elemento sea mostrado como un bloque pero que se comporte como un elemento en línea. Esto significa que el elemento ocupa solo el espacio necesario en la línea y permite establecer ancho y alto.

Para añadir display: inline-block a un elemento HTML, puedes utilizar el atributo style o definir una regla de estilo en una hoja de estilos CSS.

Por ejemplo, para hacer que un elemento sea mostrado como inline-block, puedes añadir style="display: inline-block;" al elemento HTML o definir en tu hoja de estilos:

elemento {
        display: inline-block;
      }

Otros

Combinators

Los combinators son selectores en CSS que nos permiten seleccionar elementos basándonos en su relación con otros elementos. Los combinators más comunes son los siguientes:

  • Selector de descendencia (espacio): Selecciona un elemento que es un descendiente de otro elemento.
  • Selector de hijo directo (>): Selecciona un elemento que es un hijo directo de otro elemento.
  • Selector de adyacente (o hermano siguiente) (+): Selecciona un elemento que es el siguiente hermano de otro elemento.
  • Selector de hermano general (~): Selecciona todos los elementos hermanos de otro elemento.

Pseudo-classes y Pseudo-elements

Las pseudo-clases y pseudo-elementos son selectores en CSS que nos permiten aplicar estilos a elementos basándonos en su estado o posición en relación con el árbol de documentos.

Las pseudo-clases se utilizan para seleccionar elementos en un estado específico, como :hover para aplicar estilos cuando el mouse está encima del elemento.

Los pseudo-elementos se utilizan para aplicar estilos a partes específicas de un elemento, como ::before para añadir contenido antes del contenido del elemento.

Por ejemplo, para aplicar estilos a un enlace cuando el mouse está encima, puedes usar:

a:hover {
        color: red;
      }

Y para añadir un pseudo-elemento antes del contenido de un elemento, puedes usar:

elemento::before {
        content: "►";
      }
